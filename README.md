# GraphQL FastAPI Service

A FastAPI-based server providing a GraphQL API service. This project is designed to handle task management using GraphQL queries with an optional JWT-based authentication system.

## Features
- **FastAPI backend**: Fast and efficient web framework.
- **GraphQL API**: Flexible querying with GraphQL for task management.
- **JWT Authentication**: Secure API with JSON Web Token (JWT) for authorization.
- **Modular architecture**: Clean and maintainable structure with resolvers and types.
